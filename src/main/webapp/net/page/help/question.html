1. 为什么要使用数汇宝？能否为我带来收益？

数汇宝是你的个人数据银行，能给你提供安全、高效的数据管理以及数据服务，提高你的生活品质。 你可以安心的将数据存储在数汇宝里，不用担心数据被盗或数据丢失。同时我们采用专业化的数据管理方式，为你提供便捷的数据获取、查询等服务。数汇宝在保护个人隐私的前提下，挖掘数据的价值，所得收益安装一定比例与用户共享。

2. 数汇宝是否收费? 都支持哪些平台?

数汇宝的注册用户采用免费使用的方式，获取用户个人的数据无需支付相关费用。数汇宝支持同一帐号下的多平台数据同步，目前支持的平台包括但不限于： Android Phone, Windows Phone, Java 平板电脑，PC Web app。

3. 数汇宝怎么使用

注册个人用户账号。请进入注册页面并按照我们的提示信息注册个人账号。帐号注册成功之后即为数汇宝的正式用户，相关数汇宝相关的权利及义务。也只有注册用户才能实现不同平台之间的数据同步备份，享受个人数据所带来的收益。 同步个人数据。你需要将个人数据上传到数汇宝，这些数据包括但不限以下内容：个人收支数据、个人财产数据、个人计划数据、个人人脉圈数据、个人健康数据、个人日志数据、个人文档数据以及个人感知数据等。为方便用户数据的上传，我们设置了相应的用户接口，你可以通过该接口将这些数据上传到数汇宝。 个人数据的管理。对个人数据你拥有使用权，你可以在遵循国家相关法律的基础上按照自己的需要管理这些数据。同时你也可以跟踪你的数据的交易情况，并查看相应的个人收益。 数汇宝会在保护你的个人隐私的情况下对你的数据进行处理以及交易，并支付给你相应的报酬。

4. 为什么要注册数汇宝帐号?

只有注册用户才能上传、管理数据，享受数汇宝所提供的相应服务，进行用户间的交流。

用户的注册信息是数汇宝支付报酬的依据。

5. 为什么要同步数据?怎样同步数据?

同步数据是指将不同的使用平台数据同步到云端，这样，你在任何平台对数据的修改都会在其他平台中程序出来，达到数据的透明化管理的目的。

用户登录后数据是自动同步的，无需用户额外的操作。当然用户也可以选择对特定的数据进行同步。

6. 其他平台的数据可以导入吗?

对于我们所支持的平台，你可以选择一键导入的功能将其他平台数据导入到数汇宝。

7. 数据可以备份吗?

将手机等平台上的数据同步到云端之后，可以登录我们的网站，将数据导出到电脑上，导出数据的保存格式为execl。

8. 数汇宝的收益是如何计算的？以什么方式支付？

数汇宝的收益是按照数据的价值以及使用频率等信息进行定价的，用户也可以查询到个人数据所获收益。

数汇宝的收益通过第三方进行支付，如通过支付宝等进行支付。
